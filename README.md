# Parabank

## Instalar las depentencias
```
  npm i
```

## Ejecutar desde consola
- Si es Mac:  
```
  ./node_modules/.bin/cypress open
```

- Si es Win: 
  ```
  .\node_modules\.bin\cypress open
  ```
  
## Ejecutar Pruebas con CMD o Terminal
```
  npm test
```
